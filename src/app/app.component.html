XComponent CLI

Unlesh the power of microservices. Simplified development and operation of microservice for modern distributed applications. XComponent CLI takes lets you boostrap your project quickly in time.

XComponent CLI will help you install and generate a XComponent project that gets your started in no time. We provide you multiple project template to get started with our best practices regarding XComponent development.

Features

  • Install XCStudio with all the required dependencies.
  • Initialize your XCStudio project
  • Build your components
  • Deploy and run your component to your XC Cloud Platform

Our reference guide provides full detailed informations about building component with XCStudio.

Quick Start

Follow our "Quick Start" example below to get started with XComponent development. First step involves downloading XComponent CLI which will let you install the latest XComponent development tools.

Install XC Studio with all its dependencies
$ xc install
Initialize your first project
$ xc init
Start Helloworld microservice
$ build\startMicroservice.cmd
Start Helloworld client
$ build\startConsoleApp.cmd
Open XCStudio
$ xcstudio.cmd
